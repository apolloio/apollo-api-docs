# Misc

## Get a list of users

```shell
curl -X GET -H "Content-Type: application/json" -H "Cache-Control: no-cache" "https://api.apollo.io/v1/users/search?api_key=YOUR_API_KEY_HERE"
```

`GET https://api.apollo.io/v1/users/search`
